FindBugs - A Bug Pattern Detector for Java

This is the web page for FindBugs, a program which looks for bugs in Java code.  It is free software, distributed under the terms of the Lesser GNU Public License.

FindBugs requires JRE (or JDK) 1.4.0 or later to run.  However, it can analyze programs compiled for any version of Java.

The current version of FindBugs is @VERSION@, released on @RELEASE_DATE@.

News

We will be giving a presentation about using static analysis to find bugs at JavaOne 2004.  We will also be participating in a FindBugs BOF (Birds of a Feather) session.  Hope to see you there!

Changes since version 0.7.2:

  • Documentation includes an HTML document with descriptions of all bug patterns reported by FindBugs
  • Changed the Inconsistent Synchronization detector so that fields synchronized 50% of the time (or more) are reported as medium priority bugs (previously they were reported as low)
  • New detector to find private methods that are never called
  • New detector to find suspicious uses of non-short-circuiting boolean operators (& and |, rather than && and ||)
  • FindBugs web page has a link to a Japanese translation of the FindBugs manual, contributed by Hiroshi Okugawa
