FindBugs - A Bug Pattern Detector for Java

This is the web page for FindBugs, a program which looks for bugs in Java code.

Software

The current release is 0.5.0:

  • Binary distribution
  • Source distribution

FindBugs is pre-alpha quality software.  Please be advised that

  • it contains bugs,
  • functionality may be added or removed without notice, and
  • the source code may be drastically restructured in the future

Having said that, we invite you to try it out, and especially send us feedback.

Resources

There are two mailing lists for FindBugs.

  • Findbugs-announce is a low volume list for announcements of new releases.
  • Findbugs-discuss is for discussion of planned features, bugs, development issues, etc.

We submitted a paper describing FindBugs to OOPSLA 2003.
